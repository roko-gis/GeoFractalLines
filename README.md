# GeoFractalLines
GeoFractalLines v1.0.0 — QGIS-based Python framework for fractal and multifractal analysis of vector line geometries, including adaptive box-counting, statistical validation (bootstrap &amp; CSR tests), and full multifractal spectrum reconstruction.
